# Sniff

- Packet Sniffer in GO

## build

```bash
$ make
```

## Tests

```bash
$ sudo ./tests
```

## CMD

```bash
$ sudo sniff -h
    Usage of ./sniff:
        -i string
            Network Interface (default "lo")
        -l int
            Loop quantity (default 1)
        -ls
            List of Network Interface
        -o string
            Output log file
```
